Mapping Large Data Sets

V1: The Problem

Displays the direct approach of fetching all data from the server and rendering it on the map.

V2: Filtering By View

The data retrieved from the server is limited to only what is contained in the bounding box of the current view.

V3: Server-side Clustering & Caching

The data is clustered at lower zoom levels and cached, while at higher zoom levels it is bound by the current view.

V4: Adds in Client-side Clustering

Adds client-side clustering to V3.
