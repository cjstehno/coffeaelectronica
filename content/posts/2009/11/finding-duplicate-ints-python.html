I decided to start playing around with Python a bit and figured I should do what is becoming my personal tradition of implementing the Duplicate Int Finding Problem with Python.
It's actually a very interesting language and I am surprised that I have not really looked into it sooner. It has a rich set of built-in libraries and tons of extension modules. The code I came up with is pretty straight-forward: 

def find_dup_int(items):
    items.sort()
    for i in range(len(items)-1):
        if items[i] == items[i+1]:
            return items[i]
    raise RuntimeError('No duplicate values found!')
